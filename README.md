# Сайт-визитка (репозиторий)

Небольшой статический проект: две страницы-визитки (обычная и версия с калькулятором стоимости). Без сборки, фреймворков и зависимостей.

## Что внутри

- `Сайт-визитка 2.html` — базовая страница-визитка
- `Сайт-визитка 2 калькулятор исправленный.html` — страница с блоком «Калькулятор стоимости»

Также в папке могут быть дополнительные копии (`*.txt`) — это черновые/архивные материалы.

## Как открыть локально

Просто откройте нужный `.html` файл в браузере (например, двойным кликом по файлу).

## Точка входа

`index.html` добавлен как «главная» страница репозитория: он содержит ссылки на обе версии.

## Публикация (GitHub Pages)

Если будете выкладывать на GitHub Pages:

1. Убедитесь, что в корне репозитория есть `index.html` (он добавлен).
2. В настройках Pages выберите публикацию из корня (`root`) репозитория.
3. Открывайте сайт с главной страницы `index.html`.
